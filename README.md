# 🏦 Калькулятор банковских вкладов

## О проекте

Три года назад я выполнял тестовое задание на позицию фронтенд-разработчика - нужно было сверстать калькулятор банковских вкладов. Тогда я был совсем зелёным: фиксированная ширина 1024px, `float` вместо нормальной раскладки, browser-хаки для IE и Firefox, а JavaScript-логику я так и не реализовал - кнопка «Рассчитать» вела в никуда.

Спустя 3 года я решил вернуться к этому заданию и довести его до ума - посмотреть, насколько вырос за это время. Результат перед вами.

## Что было vs. Что стало

### v1.0 (2023) - исходное тестовое задание

Что я делал тогда:
- Статичная вёрстка по макету без адаптивности
- Фиксированная ширина `1024px`, `body { width: 1024px }`
- CSS на `float`, абсолютном позиционировании и `margin`-магии
- Хаки `@-moz-document url-prefix()` и `@media (-ms-high-contrast)` для кроссбраузерности
- Отдельные классы для каждого пункта меню (`.but_menu1`, `.but_menu2`, ... `.but_menu6`)
- Нет JavaScript - калькулятор не считал
- Нет `viewport` - на мобильных всё разъезжалось
- Inline-стили

**Честная самооценка:** 3/10. Для джуниора на тот момент - нормально, но сейчас на это больно смотреть.

### v2.0 (2026) - полная переработка

Что я умею сейчас:

**HTML:**
- Семантическая разметка: `<header>`, `<main>`, `<footer>`, `<nav>`, `<section>`, `<article>`
- ARIA-атрибуты (`aria-label`, `aria-expanded`, `aria-current`) для доступности
- Правильная связка `<label>` + `for`, телефоны как `<a href="tel:">`
- `<ol>` для breadcrumbs с `aria-current="page"`

**CSS:**
- CSS Custom Properties - единая система дизайн-токенов
- CSS Grid + Flexbox для раскладки - `float` в прошлом
- Полная адаптивность: 320px → 2560px с 2 брейкпоинтами
- Sticky header с `backdrop-filter: blur()`
- Стилизованные range-слайдеры, toggle-кнопки из `radio`
- Плавные анимации и hover-эффекты
- Ноль browser-хаков - всё работает нативно

**JavaScript (ES6+):**
- Полностью рабочий калькулятор сложных процентов с формулой $FV = P(1+r)^n + PMT \cdot \frac{(1+r)^n - 1}{r}$
- Синхронизация range-слайдеров с текстовыми полями в реальном времени
- Форматирование чисел с разделителями (100 000 ₽)
- Визуальный прогресс-бар: вклад / пополнения / проценты
- Мобильное бургер-меню с overlay
- Авто-расчёт при загрузке страницы
